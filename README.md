# nd-try
React + Redux + Matrial-ui / Webpack

## set up

```shell
$ git clone git@github.com:hkwid/nd-try.git
$ npm i
```

## to run server
```shell
$ npm run api
```

## to build
```shell
$ webpack
# watch
$ webpack --watch
```

## to test
```shell
$ npm run test
```

## folder structure
```shell
├── app.js # server
├── bin  # for server
├── data # json files
├── package.json
├── public # static files
├── routes # for server router
├── src # browser
├── test # test code
├── views
└── webpack.config.js
```
